# GeoDrop Color Palette

This document defines the official color palette for GeoDrop's retro pixel art aesthetic.

## Core Terrain Colors

- **Light Dirt** - `#E0B070` - Surface-level soil, warm tan
- **Mid Soil** - `#A86C3A` - Mid-depth earth, rich brown
- **Deep Earth** - `#603824` - Deep underground, dark chocolate brown
- **Stone Layer** - `#8B7B6A` - Rocky foundation, cool gray-brown
- **Bedrock** - `#3A3A3A` - Unbreakable base layer, dark charcoal

## Gem & Resource Colors

- **Ruby Red** - `#E94E3C` - Rare red gems, vibrant crimson
- **Sapphire Blue** - `#3B9EE9` - Precious blue gems, bright sapphire
- **Emerald Green** - `#3CC86B` - Valuable green gems, rich emerald
- **Topaz Yellow** - `#F7C443` - Golden yellow gems, warm amber
- **Amethyst Purple** - `#A76CF4` - Mystical purple gems, deep violet
- **Diamond White** - `#E8F4F8` - Ultimate treasure, brilliant white
- **Coal Black** - `#2B2B2B` - Common fuel resource, deep black
- **Iron Gray** - `#B0B0B0` - Metal ore, neutral silver-gray
- **Gold** - `#FFD700` - Precious metal, classic gold

## Background & Depth Layers

- **Far Earth Shadow** - `#4A3429` - Deep parallax background, dark brown
- **Cave Mist** - `#7A6D73` - Atmospheric depth effect, dusty mauve
- **Sky Gradient Top** - `#87CEEB` - Surface sky, light blue
- **Sky Gradient Bottom** - `#FFD07A` - Horizon glow, warm peach

## UI & Highlight Accents

- **Gold Highlight** - `#FFCB4A` - Score/currency indicator, bright gold
- **Teal Accent** - `#2EC5B6` - Interactive elements, vibrant teal
- **Warning Orange** - `#FF8C42` - Alerts/danger zones, safety orange
- **Health Red** - `#D43C3C` - Health/damage indicators, alert red
- **Text Primary** - `#F5F5F5` - Main UI text, off-white
- **Text Secondary** - `#B0B0B0` - Subtle labels, light gray
- **UI Background** - `#2A2A2A` - Panel backgrounds, dark charcoal
- **UI Border** - `#4A4A4A` - Panel borders, medium gray

## Usage Guidelines

- Maintain high contrast between interactive elements and backgrounds
- Use warm earth tones for terrain to create depth perception
- Reserve vibrant gem colors for rare collectibles
- Apply UI colors consistently across all interface elements
- Consider accessibility when choosing text/background combinations
